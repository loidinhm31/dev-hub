# Codebase Summary

This document provides a high-level summary of the Dev-Hub codebase based on the latest automated compaction.

## Overview
Dev-Hub is a workspace management system designed for agent-based development, featuring a built-in file explorer, terminal manager, and integrated editor.

## Key Features
- **IDE Explorer:** Provides a file tree, editor, and workspace navigation capabilities.
- **Agent Store:** A central repository for agent configurations, skills, and commands.
- **Workspace Protocol:** Defines communication between agent sub-systems and the filesystem.

## Configuration Defaults
- Environment variable overrides may be used for certain behaviors.

## Architecture
- `server/`: Rust backend handling core FS operations, WebSocket transport, and agent store management.
- `packages/web/`: React frontend providing the user interface, utilizing TanStack Query for data synchronization.
- `docs/`: System architecture, API, and configuration documentation.
